## 🚀 使用方法

### 基本使用
```
1. 放置告示牌
2. 在第一行写: example.com/
3. 在第二行写: image.png
4. 图片会自动显示
```

### 带属性
```
1. 第一行: example.com/image.png
2. 第二行: x2  (高度x2)
3. 第三行: Y90 (Y轴旋转90度)
```

### 命令
```
/signpic - 帮助
/signpic version - 版本
/signpic reload - 重载配置
/signpic clear - 清除缓存
```

## 🎓 技术决策

### 已简化的部分
1. **ASM → Mixin**: 更现代、更安全
2. **复杂的Compat层 → 直接API**: 单版本支持
3. **bnnwidget → 原生实现**: 减少依赖
4. **复杂的属性解析 → 简化版本**: 够用即可

### 保留的核心
1. **图片加载和渲染**: 完整实现
2. **GIF支持**: 完整实现
3. **缓存系统**: 完整实现
4. **状态管理**: 完整实现
